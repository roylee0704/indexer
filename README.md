# indexer
Indexer, an experimental project in Golang.


## Functional Requirements
1. **Parse** a file!
  - text file at the moment.

2. **Split** strings of words!
  - build hit-item for each word.
  - build word-frequency.
  - goal: fast building construction.

3. **Index** it!
  - persistent it.
  - goal: fast insertion.
